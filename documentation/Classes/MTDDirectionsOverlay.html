MTDDirectionsOverlay Class Reference

Inherits from NSObject
Conforms to MKOverlay
Declared in MTDDirectionsOverlay.h

Overview

An instance of MTDirectionsOverlay implements the MKOverlay protocol and represents an overlay used to store information about already parsed directions with a given fromCoordinate and toCoordinate. It stores all waypoint along the route and the total distance of the route.

This category exposes private API used in MTDirectionsKit to other classes to not generate warnings when using them. You should not use the properties and methods exposed here directly in your code.
