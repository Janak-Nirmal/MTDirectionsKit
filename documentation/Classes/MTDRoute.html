Inherits from NSObject
Declared in MTDRoute.h

Overview

An instance of MTDRoute represents one way of travelling from a starting point to an end point. When requesting directions alternative routes can be provided, each of whom is represented by an instance of MTDRoute.

This category exposes private API used in MTDirectionsKit to other classes to not generate warnings when using them. You should not use the properties and methods exposed here directly in your code.
