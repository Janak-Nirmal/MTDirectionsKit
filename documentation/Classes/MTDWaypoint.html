Inherits from NSObject
Conforms to NSCopying
Declared in MTDWaypoint.h

Overview

An instance of MTDWaypoint is a lightweight object wrapper for either a CLLocationCoordinate2D coordinate or an address string representing a location (or both).

It is used in MTDDirectionsKit to store coordinates in collections like NSArray.
