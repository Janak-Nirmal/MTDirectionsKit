Inherits from GMSPolyline

Overview

An instance of MTDGMSDirectionsOverlayView is a subclass of GMSPolyline and is used to draw the directions/route on top your instance of MTDGMSMapView (Google Maps SDK). It draws a path including all waypoints stored in one route of the underlying instance of MTDDirectionsOverlay.

Therefore this class behaves different from its analog class in the MapKit version of MTDirectionsKit as it is only responsible for drawing one route and not several alternative routes.

Remark: This class currently cannot be customized using UIAppearance.
