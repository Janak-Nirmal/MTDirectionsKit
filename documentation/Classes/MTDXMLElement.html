MTDXMLElement Class Reference

Inherits from NSObject
Declared in MTDXMLElement.h

Overview

An instance of MTDXMLElement represents one XML node of a whole XML tree. It is used to parse XML documents and uses libxml under the hood, which allows for convenient use of XPath.
