MTDDirectionsParser Protocol Reference

Conforms to NSObject
Declared in MTDDirectionsParser.h

Overview

The MTDDirectionsParser protocol defines a set of required methods that instances of MTDDirectionsParser need to implement, that want to act as a directions parser.
